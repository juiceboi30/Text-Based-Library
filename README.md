# Text-Based-Library
This a Simple Text based Library for Java Beginners. It teaches you Data Structs, Array, Class, Inheritance, Encapsulation etc.... Learn More...!
1️ The program creates a Library object that stores books in an ArrayList.
2️ Book objects hold data (title, author, borrowed status).
3️ User is a base class, with Student and Teacher inheriting and implementing borrowing behavior.
4️ The user can add, remove, find, borrow, and return books through library methods.
5️ Main connects everything by creating objects and calling methods to run the app.
